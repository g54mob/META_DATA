## Process

1. Read all phases, scripts, and docs in `LEARN/{PROJECT}/`
2. Read all workflow files in `.windsurf/workflows/`
3. Compare: do workflow instructions match what actually happens during builds?
4. Identify stale instructions:
   - Steps that no longer apply
   - Missing steps that should be added
   - Incorrect file references
   - Outdated checklist items
5. Update workflow files to match current reality
6. Report what changed
